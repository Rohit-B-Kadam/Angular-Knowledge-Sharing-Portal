What is Knowledge Sharing Portal?

It is Angular based project. In this site you can see number of projects list by other people. You can also add your own project that you want to shared with others.

Technique or Technologies used in this project ?

  • All the basic Angular concept
  • Service (dependence injection)
  • Routing
  • Inter component cummuication
  • Dynamic UI
  • Bootstrap4 and ngBootstrap
  • Web site is fully responsive on desktop as well as on mobile
  • FireBase Hosting

Future Plan

  • Firestore Cloud(real time data)
  • Firebase Authenication
